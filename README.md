# PCD-ESCOM
Programacion para Ciencia de Datos course at ESCOM
---
Repositorio de retos semanales del curso. Cada reto resuelve un problema de procesamiento de datos, aplicando los conceptos vistos en clase de forma progresiva.

## Estructura
Cada reto se encuentra en una carpeta llamada ```reto-semana-<número de semana>```, mientras que las evaluaciones se organizan por parciales. Cada sección contiene el programa principal ```main.py``` y su respectiva documentación.
