# PCD-ESCOM
Programacion para Ciencia de Datos course at ESCOM
---
Repositorio de retos semanales del curso. Cada reto resuelve un problema de procesamiento de datos, aplicando los conceptos vistos en clase de forma progresiva.

## Estructura
Cada reto se encuentra en una carpea llamana ```reto-semana-<numero de semana>``` en el cual se encuentra el programa principal ```main.py``` con su respectivo archivo ```README.md``` en donde se describe todo acerca de cada reto.
